# Changelog

## 2.2.0

### Changed

- Raised supported runtime to Node.js `>=24`.
- Updated dependencies and devDependencies, including `qs`, `eslint`, `jest`,

- Migrated local quality tooling to current setup (`husky` v9 and `lint-staged`).
- Removed legacy CI/pipeline leftovers and updated automation-related configs.

### Fixed

- Resolved lint issues after the ESLint v9/config upgrade.
- Updated and cleaned up tests and repository metadata.
